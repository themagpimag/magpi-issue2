How to start a graphical interface

 

So you've followed the guide written by meltwater, and you've got your Debian operating system installed.  You have booted up, logged in and are now looking at a flashing cursor thinking, “Now what?”.  Let's start up a windows-like interface that you may be more comfortable with.

 

At that command prompt, type:

 

startx

 

And press enter.  Be patient, it may take a minute or two to load, but you should now have a graphical desktop.  You can wiggle that mouse with glee if you so desire.

Shutting down your Raspberry Pi

 

If you haven't started the graphical interface and are looking at the terminal screen, simply type:

 

sudo shutdown -h now

 

And press enter.

 

If you have started the graphical interface with the startx command, then click the little    icon (in the bottom right hand corner).  Then choose the shutdown option from the menu that appears.
