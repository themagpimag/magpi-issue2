Synaptic - The friendly way to install software

 

You will be aware by now that we can install software with that cheeky apt-get command, but suppose you want a more user-freindly way of doing this?  Something easier than typing at a command line.  Let's install Synaptic package manager.

 

If you have not already done so, start the the graphical environment as explained previously:

 

startx

 

When that's loaded, click the start menu (bottom left hand corner) and choose Accesories → LXTerminal.  Now type:

 

sudo apt-get install synaptic

 

And press enter.  Yes I know it's another command line command, but this part is necessary.

 

You will be prompted for your password, type this in and press enter.  This kicks off the install process.  You may have to answer yes if prompted (just type y and press enter).  After a bit of time, your prompt will come back (this took about 10 minutes on my Raspberry Pi, so be patient!).

 

Type:

 

exit

 

And press enter.

 

Please bear in mind that the next few steps may take some time.  The Raspberry Pi is amazing, but we are asking a lot of it here!
