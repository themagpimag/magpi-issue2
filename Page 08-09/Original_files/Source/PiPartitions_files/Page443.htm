When I did it previously I had a few issues on my RPi, no idea if you had them same or not:

a)

To Run synaptic, it needed an admin password...but login password fails...

In the end I found this...

Workaround for the admin password issue (apparent bug in distro):

 

"gconftool-2 -\-type bool -\-set /apps/gksu/sudo-mode true"

 

This tells it to use sudo rather than su account.

b) Rebuild of search index requires lots of memory.  If you have non-standard RAM split (i.e.XBMC distros) it crashes.

free -m           says what memory you have (122Mb total for the XBMC install)

 

To switch (select one and reboot):

sudo cp /boot/arm192_start.elf /boot/start.elf- to get 192MB

sudo reboot

 

To switch back:

sudo cp /boot/arm128_start.elf /boot/start.elf- to get 128MB

sudo reboot
